// Sobre el proyecto

Seguridad Integral Automatizada

CyberShield Pro unifica módulos ofensivos y defensivos bajo una misma plataforma, orquestados por agentes de IA a través de n8n. Diseñado como proyecto de fin de grado para demostrar que la ciberseguridad puede ser proactiva, inteligente y accesible.

// Módulo Ofensivo

Ataque y Penetración

Catálogo completo de vectores de ataque organizados por categoría. Cada ataque se configura con parámetros específicos y se ejecuta remotamente via webhooks de n8n hacia Kali Linux.

🌐
CAPA 2

Infraestructura LAN

MAC Flooding, Port Stealing, SPAN/Mirror, túneles encubiertos DNS/ICMP para exfiltración de datos.

📡
WIRELESS

Auditoría Wireless

Des-autenticación, Evil Twin, captura de handshake WPA2, ataques WPS Pixie Dust, PMKID clientless.

🐍
SCAPY

Laboratorio Scapy

Crafting de paquetes, escaneo de vulnerabilidades, scripts de automatización encadenados con n8n.

// Módulo Defensivo

Detección y Protección

Monitorización en tiempo real con múltiples capas de defensa. Integración con Wazuh para SIEM y reglas personalizadas de detección de amenazas.

🔥

5 Firewalls Activos

Palo Alto, Fortinet, Check Point, Cisco Firepower y Sophos XGS monitorizados en tiempo real con métricas de throughput y uptime.

📊

Monitor de Tráfico 24h

Visualización de tráfico entrante vs bloqueado. Detección de patrones anómalos con alertas automáticas por severidad.

🛡️

Reglas de Protección

DDoS mitigation, IPS/IDS, WAF, Geo-Blocking, Rate Limiting, SSL Inspection, DNS Filtering y Sandbox Analysis.

🌍

Mapa de Amenazas

Geolocalización de origen de ataques. Asia-Pacífico lidera con el 38% de las amenazas detectadas, seguida de Europa del Este.

📋

Compliance ISO

Seguimiento de cumplimiento ISO 27001, 27002, 27005 y NIST CSF con indicadores de progreso y auditorías automatizadas.

🔍

Wazuh SIEM

Integración completa con Wazuh para correlación de eventos, reglas custom de detección y respuesta automatizada a incidentes.
